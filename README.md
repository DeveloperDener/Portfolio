# Portfólio de Dener Caetano

Este é o site pessoal de Dener Caetano, desenvolvido com foco em Front-End. O projeto apresenta um design moderno, responsivo e interativo, utilizando HTML5, CSS3 e JavaScript.

## Funcionalidades

- Layout responsivo que se adapta a diferentes tamanhos de tela
- Menu de navegação funcional entre as páginas
- Formulário de contato com validação em JavaScript
- Efeitos interativos para melhorar a experiência do usuário
- SEO básico aplicado para melhor indexação

## Tecnologias Utilizadas

- HTML5
- CSS3 (Flexbox, Media Queries)
- JavaScript (Validação de formulário, animações)
- Google Fonts (Montserrat, Bebas Neue)
